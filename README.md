Habby is a simple web app that helps users track their daily habits and understand their health patterns over time.

Features

User registration and login

Save user data in a local JSON database

Log daily habits

Edit logged days

View daily results

View weekly results

Personalized feedback based on age and gender

Clean and minimal UI using Streamlit

Tracked Habits

Sleep hours

Water intake

Steps walked

Mood

Study / work hours

Hobby time

Tech Stack

Python

Streamlit

Plotly

JSON (local storage)

How to Run

Clone the repository

Install requirements

Run the app
pip install -r requirements.txt
streamlit run habby.py
